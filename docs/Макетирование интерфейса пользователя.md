## Логика взаимодействия

1.  **Навигация:** Пользователь перемещается между разделами «Создать», «Сканер» и «История» с помощью нижней панели (Bottom Navigation Bar).
2.  **Создание кода:**
    *   На главном экране выбирается тип данных (например, Wi-Fi).
    *   Заполняются поля и выбирается цвет.
    *   При нажатии «Сгенерировать» открывается экран результата с готовым изображением.
3.  **Сканирование:**
    *   При открытии вкладки «Сканер» активируется камера.
    *   При обнаружении кода сканирование происходит автоматически.
    *   Результат показывается во всплывающем окне или на отдельном экране.
4.  **История:** Все действия автоматически сохраняются. Пользователь может нажать на элемент списка истории, чтобы снова открыть результат.
