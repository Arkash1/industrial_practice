# Обоснование выбора стека технологий

Для разработки приложения **QR Code Generator** был выбран следующий стек технологий:

- **Язык программирования:** Java  
  Надежный и строго типизированный язык, являющийся классическим стандартом для нативной Android-разработки.

- **Платформа:** Android  
  Обеспечивает мобильность устройства, необходимую для использования камеры при сканировании кодов в реальном мире.

- **Библиотеки работы с QR:** ZXing и ML Kit  
  **ZXing** используется для генерации изображений благодаря своей стабильности. **ML Kit** (в связке с CameraX) выбран для сканирования, так как обеспечивает высочайшую скорость распознавания.

- **ORM:** Room (SQLite)  
  Библиотека из состава Android Jetpack, упрощающая работу с локальной базой данных для хранения истории и избранного.

- **Архитектура:** MVVM  
  Разделяет логику отображения и обработки данных, что делает код чистым, тестируемым и устойчивым к изменениям конфигурации экрана.

- **Инструменты разработки:** Android Studio, Gradle  
  Официальная интегрированная среда разработки, предоставляющая мощные инструменты для отладки UI и профилирования производительности.

Выбранный стек технологий соответствует требованиям к высокой скорости обработки изображений и полной автономности (работе офлайн).
