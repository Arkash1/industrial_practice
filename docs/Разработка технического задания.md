
# Разработка технического задания

Целью разработки является создание **мобильного приложения «Генератор QR-кодов»**, предназначенного для создания, кастомизации и считывания матричных штрихкодов (QR-кодов).  
Программный продукт должен позволять пользователю быстро генерировать коды для различных типов данных (текст, ссылки, настройки Wi-Fi), настраивать их визуальное оформление, а также распознавать информацию с физических носителей.

Разработка приложения направлена на **упрощение обмена цифровой информацией** между пользователями и физическими объектами, а также на объединение функций сканера и генератора в едином инструменте.

## Характеристика программного продукта

Разрабатываемый программный продукт представляет собой **мобильное приложение для операционной системы Android**.  
Приложение обеспечивает:

- Возможность **генерации QR-кодов** для 8 типов данных (Текст, URL, Контакты, Wi-Fi и др.)
- **Кастомизацию** внешнего вида кодов (изменение цветов)
- **Сканирование** кодов в реальном времени с использованием камеры устройства
- Автоматическое **сохранение истории** созданных и отсканированных кодов
- **Экспорт изображений** в галерею и их отправку через сторонние приложения

Система включает в себя следующие основные компоненты:

- **Пользовательский интерфейс** на базе Material Design 3
- **Модуль генерации** (на основе библиотеки ZXing)
- **Модуль сканирования** (с использованием CameraX и ML Kit)
- **Модуль хранения данных** (локальная база данных Room)

Приложение работает **полностью автономно** и не требует обязательного подключения к сети Интернет для выполнения основных функций.

## Целевая аудитория

Целевой аудиторией приложения являются пользователи мобильных устройств на базе Android, которым необходим инструмент для бесконтактного обмена информацией.  
Приложение ориентировано на:

- **Рядовых пользователей** (обмен контактами, ссылками, настройка Wi-Fi для гостей)
- **Представителей малого бизнеса** (создание кодов для визиток, меню, рекламных материалов)
- **Офисных сотрудников и студентов** (быстрый обмен документами и ссылками)

Для работы с приложением не требуется специальных технических знаний — интерфейс рассчитан на пользователей с базовыми навыками владения смартфоном.

## Общие требования к программному продукту

К разрабатываемому приложению предъявляются следующие требования:

- **Удобный и современный пользовательский интерфейс**
- **Высокая скорость генерации и распознавания кодов**
- **Стабильная работа приложения** при различных условиях освещения (для сканера)
- **Корректная обработка ошибок** (например, при вводе невалидных данных)
- **Обеспечение конфиденциальности** (отсутствие передачи данных на внешние серверы)
- **Возможность расширения списка поддерживаемых типов данных** в будущем
