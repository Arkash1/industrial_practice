# Построение диаграммы вариантов использования


<img src="https://github.com/Arkash1/industrial_practice/blob/master/docs/diagram.png" alt="Диаграмма вариантов использования" width="600" />

## Акторы системы
1. **Пользователь** — инициатор взаимодействия, который вводит данные для генерации или использует камеру для сканирования.
2. **Система** — внутренние процессы приложения, отвечающие за обработку изображений, валидацию данных и сохранение истории.

## Основные варианты использования
- Генерация QR-кода (Текст, URL, Wi-Fi, Контакты)
- Сканирование QR-кода через камеру
- Настройка внешнего вида (кастомизация цветов)
- Сохранение сгенерированного кода в галерею
- Автоматическое сохранение в историю операций
- Поделиться кодом через внешние приложения
